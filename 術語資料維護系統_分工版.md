# 術語資料維護系統 - 分工版

## 一、後端（Backend）
- 資料表與遷移
  - Term、TermLanguage、TermAlias、Category、TermVersion
- API 端點
  - 術語 CRUD
  - 分類 CRUD 與排序
  - 語言內容更新
  - 搜尋/過濾/排序
  - 批次操作
  - 匯入/匯出（含預覽）
  - 版本紀錄與差異比對
- 驗證與正規化
  - 大小寫規則、trim、多空白合併
  - 重複術語/別名檢查
- 回應格式與錯誤碼
- 匯入作業狀態追蹤

## 二、前端（Frontend）
- 列表頁
  - 搜尋、過濾、排序、分頁
  - 欄位顯示自訂與固定欄
  - 勾選與批次操作
- 新增/編輯
  - 基本欄位
  - 動態語言欄位
  - 別名/同義詞
  - 說明/備註
- CSV 匯入
  - 上傳、欄位映射、預覽、結果
- CSV 匯出
- 版本紀錄與差異比對
- 空狀態/錯誤提示/載入狀態

## 三、QA
- 測試案例設計
  - CRUD 正常流程
  - 多語欄位動態新增
  - 搜尋/過濾/排序/分頁
  - 批次操作
  - 匯入/匯出（含錯誤）
  - 版本紀錄與差異比對
- 驗收標準檢核
- 回歸測試與問題追蹤
